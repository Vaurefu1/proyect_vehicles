# proyect_vehicles

Esta aplicación web permite explorar un conjunto de datos de vehículos usados.
Si algún usuario quiere interactuar con la información visual (gráficas), tiene acceso a botones que generan las mismas.

El desarrollo de la app esta basada en Phyton y Streamlit, desplegada en Render.

La aplicación permite:

- Visualizar un histograma del kilometraje de los vehículos.
- Visualizar la relación entre el precio y el kilometraje.
- Se interactúa con los datos a través de botones en la interfaz.

Tecnologías aplicadas:

- Python
- Pandas
- Streamlit
- Plotly
- Render
